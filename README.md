# Google Web Fonts
Most common Google Fonts used by Next.JS Devs to embed them directly.
</br>
This is made for Devs based in Germany, as a response to Lawsuits for linking to Google without a users consent.
</br>
Fonts included: Geist, Geist Mono, Public Sans, Noto Sans & Roboto.
